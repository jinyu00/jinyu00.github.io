正文

漏洞位于 upnp 服务处理 升级的流程中，用于设备升级的 upnp 服务 xml 配置文件为 etc/upnp/DevUpg.xml.

其中在获取 NewDownloadURL 和 StatusURL 后拼接命令，调用了 system 执行了。
